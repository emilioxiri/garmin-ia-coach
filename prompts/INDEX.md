# Prompts — Índice

## Carpetas

### `/tools` — Test prompts para los 10 tools
Ejemplos de conversación natural para probar cada función del coach.

- 📌 Cada archivo contiene **5 ejemplos** de prompts para ese tool
- 🎯 Todos redactados como preguntas naturales (español)
- 📋 Incluyen la estructura JSON del tool y respuesta esperada

**Inicio rápido:**
1. Ve a `/tools`
2. Lee el `README.md` para overview
3. Abre cualquier archivo numérado `01_*.md` a `10_*.md`
4. Copia un ejemplo
5. Envía al bot en Telegram

---

## Notas

- **Idioma:** Español (nativo del sistema)
- **Entrada:** Mensajes naturales
- **Procesamiento:** El LLM extrae parámetros automáticamente
- **Ejecución:** El tool correspondiente corre con esos parámetros

Ver `tools/README.md` para lista completa de tools.
